## Anti-Cannibalization Rules

1. **Only one page targets each primary keyword** - if two pages compete for the same term, consolidate or differentiate
2. **Internal links use keyword-rich anchor text** - link to pages using their primary keywords, not "click here"
3. **New content checks this document first** - before publishing a new page or article, verify its target keywords don't overlap with existing pages
4. **Review quarterly** - search rankings shift; update this strategy as pages are added or content changes
